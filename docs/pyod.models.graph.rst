Graph Detectors
================

PyOD has 8 graph anomaly detectors built on PyTorch Geometric (``pip install pyod[graph]``). All transductive in v1. Rankings from `BOND <https://arxiv.org/abs/2206.10071>`_ (NeurIPS 2022).

See :doc:`examples/graph` for usage and input format.
