Text and Image Detectors
=========================

PyOD's :class:`~pyod.models.embedding.EmbeddingOD` chains foundation model encoders (sentence-transformers, OpenAI, HuggingFace) with any PyOD detector for text and image anomaly detection. Rankings from `NLP-ADBench <https://github.com/USC-FORTIS/NLP-ADBench>`_.

See :doc:`examples/embedding` for usage.
